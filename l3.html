④viewを作成
bladeのテンプレートが使える様にresources/views/にviewのファイル名の拡張子の前に「.blade」を付けて作成する。
もしくは
ターミナル
touch resources/views/hello.blade.php
// touch resources/views/任意のファイル名.blade.php
作成後、htmlなどのデータを入力
app/Http/Controllers/HelloController.php
class HelloController extends Controller
{
      public function index(){
        $str='こんにちは。こんばんわ';
        $str2='おげんきですか？';
        return view('hello',compact('str',str2''));
 // キーを読み込む記述をする場合compactの配列として記述する
      }
    }
resources/views/hello.blade.php
<p>{{$str}}</p>
<p>{{$str2}}</p>
// キーを読み込む記述をする場合は、
// <p>{{$キー}}</p>と記述
