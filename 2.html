①web.php
末尾に追記
routes/web.php
Route::get('hello', 'App\Http\Controllers\HelloController@index');
　　　　　　　　URLになる　　　　　　　　　　　　　　　　　 コントローラー名 　　ファンクション名
②コントローラーを作成
ターミナル
php artisan make:controller HelloController
　　　　　　　　　　　　　 　　　　　　　コントローラー名　　
Enterして実行
③viewを設定する
HelloController.php が作成されるので
　　　　　　　　　　　 　 　　　　　　　←コントローラー名

　　　　　　　　　　　　　　URLになる
app/Http/Controllers/
class Controller extends BaseController
{
    public function index(){
        return view('hello');
    }
}
と追記
キーを設定する場合は
ターミナル
Return view(ʻhelloʼ, compact(ʻstrʼ , ʼstr2ʼ));
compactの配列として記述する
④viewを作成
bladeのテンプレートが使える様にresources/views/にviewのファイル名の拡張子の前に「.blade」を付けて作成する。
もしくは
ターミナル
touch resources/views/[ 任意のファイル名 ].blade.php
作成後、htmlなどのデータを入力
キーを読み込む記述をする場合は、
view(php)
<p>{{$キー}}</p>
のように記述する
⑤簡易サーバーを起動してプロジェクトをプレビュー
ターミナル
php artisan serve
と入力しEnter
Ctrl＋c で終了
