プロジェクト内での作業

参照）
https://qiita.com/ekzemplaro/items/c74e7431a8a90e2feb03

①web.php
　末尾に追記
routes/web.php
Route::get('hello', 'App\Http\Controllers\HelloController@index');

②コントローラーを作成

ターミナル
php artisan make:controller HelloController
　Enterして実行

③viewを設定する

　HelloController.php が作成されるので

app/Http/Controllers/
class Controller extends BaseController
{
    public function index(){
        return view('hello');
    }
}
　と追記
　キーを設定する場合は
ターミナル
Return view(ʻhelloʼ, compact(ʻstrʼ , ʼstr2ʼ));
　compactの配列として記述する

④viewを作成

　bladeのテンプレートが使える様にresources/views/に
　viewのファイル名の拡張子の前に「.blade」を付けて作成する。
　もしくは

ターミナル
touch resources/views/任意のファイル名.blade.php
　作成後、htmlなどのデータを入力
　キーを読み込む記述をする場合は、
viewのphp
<p>{{$キー}}</p>
　のように記述する

⑤プレビューサーバーを起動

ターミナル
php artisan serve
　と入力しEnter
<簡易サーバーを起動してプロジェクトをプレビュー>

　と入力してEnter
　Ctrl＋c で終了
