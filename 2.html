プロジェクト内でページの作成
参照）https://qiita.com/ekzemplaro/items/c74e7431a8a90e2feb03
①web.php
末尾に追記
routes/web.php
Route::get('hello', 'App\Http\Controllers\HelloController@index');
          // ↑URLになる               コントローラー名↑      ↑ファンクション名
　　　　　　　　URLになる　　　　　　　　　　　　　　　　　 コントローラー名 　　ファンクション名
②コントローラーを作成
ターミナル
php artisan make:controller HelloController
                            // ↑コントローラー名
Enterして実行
③view(実際に見るページ)を設定する
HelloController.php が作成されるので
app/Http/Controllers/HelloController.php
class Controller extends BaseController
{
    public function index(){
        return view('hello');
                    // ↑URLになる
    }
}
と追記
キーを読み込む記述をする場合
app/Http/Controllers/HelloController.php
        return view('hello',compact('str',str2''));
                        // compactの配列として記述する
resources/views/hello.blade.php
<p>{{$str}}</p>
<p>{{$str2}}</p>
<!-- キーを読み込む記述をする場合は、
    <p>{{$キー}}</p>と記述 -->
