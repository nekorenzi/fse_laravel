■課題
同じプロジェクトを使用して名刺管理のページを作成する
使用データ : namecards.sql
仕様書 : テーブルレイアウトサンプル.xlsx
●使用するデータベースの情報をインポートする
MySQLデータベースを開き、freeDBを選択した状態で「インポート」
課題データの「namecards.sql」を選択してページ一番下の「インポート」
表示されたページを確認しインポートできているか確認

●名刺トップページを作成
直接のルートをコメントアウトにして下記を追記
app/Http/Controllers/HelloController.php
Route::get('/', 'App\Http\Controller@index')->name('index');
MenuControllerを作成
ターミナル
php artisan make:controller MenuController
app/Http/Controllers/MenuController.php
    public function index
        return view('index');
    }
resources/views/にindex.blade.phpを作成
app/Http/Controllers/MenuController.php
<h1>名刺管理&lt/h1>
<br>
<p>
    <a href={{ route('cardList',[],false) }}">名刺管理一覧を開く</a>
</p>
