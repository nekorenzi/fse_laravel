■ターミナルで以下を実行
1. composer をインストール、更新 
composer install
composer update
composer dump-autoload
2.npm パッケージをインストール 
npm i or npm install
yarn i or yarn install
// ↑不要?
3.env ファイルの作成 
cp .env.example .env
$newValue = Read-Host "Enter the new value for DB_DATABASE"
                  (Get-Content .env) | ForEach-Object {$_ -replace '^DB_DATABASE=.*', "DB_DATABASE=$newValue"} | Set-Content .env
Enter the new value for DB_DATABASE: と出たら、使用するデータベース名を入力しEnter
4. マイグレートしてデータベースと同期 
php artisan migrate
5. キーの生成 
php artisan key:generate
6. キャッシュの削除 
php artisan cache:clear
php artisan config:clear
php artisan route:clear
php artisan view:clear
php artisan session:clear
--- ここからは不要? 
php artisan make:migration prescription_detail_veiw
8. npm run watch 
npm run watch
9. php artisan serve 
php artisan serve
clear check
10. php artisan optimize 
php artisan optimize
